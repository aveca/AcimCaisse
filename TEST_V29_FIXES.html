1. Panier Toujours Visible

Problème: Le panier disparaissait après scan

  1. ✅ Ouvrir AcimCaisse dans un nouvel onglet
  2. ✅ Attendre que l'application charge complètement
  3. ✅ Vérifier que le champ code-barres clignote en orange
  4. ✅ Taper un code produit (ex: 123456789)
  5. ✅ PASS: Le panier apparaît en bas de l'écran
  6. ✅ FAIL: Le panier n'est pas visible → Vérifier l'injection shadow DOM

2. Autofocus sur le Champ Code-Barres

Problème: "Autofocus processing was blocked"

  1. ✅ Recharger la page
  2. ✅ Vérifier que le champ code-barres a le focus automatiquement
  3. ✅ PASS: Le champ est focalisé et prêt à la saisie
  4. ✅ FAIL: Le focus n'est pas automatique → Vérifier forceFocusOnBarcodeInput()

3. Correction des Stocks Négatifs

Problème: Produits avec stockQty < 0

  2. ✅ Exécuter: window.AcimCaisseStockFix.check()
  3. ✅ Vérifier que negative: 0
  4. ✅ PASS: Aucun stock négatif
  5. ✅ FAIL: Des stocks négatifs existent → Exécuter window.AcimCaisseStockFix.fix()
