📝 Notes Techniques

Injection Shadow DOM

Flutter utilise un shadow DOM qui bloque les styles CSS externes. La solution:

function injectStylesIntoFlutterShadowDOM() {
  const flutterView = document.querySelector('flutter-view');
  if (flutterView && flutterView.shadowRoot) {
    const style = document.createElement('style');
    style.id = 'acim-ux-fix-style';
    style.textContent = `
      #acim-ticket-overlay {
        position: fixed !important;
        bottom: 20px !important;
        left: 50% !important;
        transform: translateX(-50%) !important;
        z-index: 9999 !important;
      }
    `;
    flutterView.shadowRoot.appendChild(style);
  }
}

Cette fonction est appelée toutes les 200ms pendant 10 secondes pour s'assurer que les styles sont injectés dès que Flutter est chargé.
